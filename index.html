Vanilla JavaScript

Vanilla JS is plain JavaScript without frameworks or libraries. It’s used for building simple, fast websites by directly manipulating HTML, CSS, and the DOM.

React

React is a JavaScript library developed by Facebook, designed for building user interfaces (UI), especially single-page applications (SPAs). It uses a component-based architecture, meaning the UI is broken into reusable, self-contained components that manage their own state.

Node.js

Node.js is a runtime environment that lets you run JavaScript outside the browser, typically on the server. It uses Google’s V8 engine and allows developers to build scalable, fast, and lightweight server-side applications.

Mongodb

MongoDB is a NoSQL database that stores data in flexible, JSON-like documents called BSON (Binary JSON). Unlike traditional SQL databases that use tables and rows, MongoDB uses collections and documents, making it great for handling large volumes of unstructured or semi-structured data.
